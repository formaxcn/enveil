## 优化方案

### 布局调整

1. **第一行**: Cloud Provider + Environment Name（两列）
2. **第二行**: SAML URL（整行）
3. **Template Values 区域**（替换原来的 Template Auto-filled）:
   - 顶部显示 Console Pattern + Account URL（两列，可编辑）
   - 下方有折叠的「高级设置」，展开后显示：
     - Account Selection Selectors (accountContainers, roleElements)
     - Console Selectors (accountContainers, roleElements)

### 需要修改的文件

1. `entrypoints/options/types.ts` - 添加 CUSTOM provider
2. `utils/cloudTemplates.ts` - 添加 CUSTOM 模板和辅助函数
3. `components/AddCloudEnvironmentModal.tsx` - 重构表单布局和交互逻辑

### UI 结构


### 交互逻辑
- 选择预设 Provider → 填充所有模板值
- 修改任意模板值 → Provider 自动变为 Custom
- 高级设置默认折叠，点击展开/收起
