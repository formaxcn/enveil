# Product Overview

**Enveil** is a Chrome browser extension designed for developers and DevOps teams to visually distinguish between different environments (Local, Staging, Production).

## Core Purpose
- Prevent accidental actions on production environments
- Provide immediate visual context about which environment you're working in
- Support flexible URL matching patterns for complex deployment scenarios

## Key Features
- **Visual Indicators**: Corner ribbons and full-page overlays
- **Flexible Matching**: Domain, URL prefix, exact URL, or regex patterns
- **Privacy-First**: 100% local configuration, no external data collection
- **Lightweight**: Minimal performance impact on browsing

## Target Users
- Web developers working across multiple environments
- DevOps engineers managing deployments
- QA teams testing across different stages
- Anyone who needs environment awareness while browsing

## Current Status
Beta / Active Development - Core functionality implemented, UI polish and testing in progress.
