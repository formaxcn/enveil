# Enveil - Chrome Extension

A Chrome extension to distinguish different environments by adding banners and overlays.

## Features

1. Add customizable banner to website corners to differentiate environments
2. Add overall overlay mask with customizable color and opacity
3. Configuration page for setting domain patterns, colors, positions, and toggle switches
4. Import/export configuration as JSON files

## Installation

1. Clone this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this directory

## Usage

Click the extension icon to open popup, then click "Configuration Settings" to configure the extension.

In the configuration page, you can:
- Set domain matching patterns and keywords
- Customize banner and overlay colors
- Toggle banner and overlay on/off
- Set banner text and position
- Import/export configurations as JSON files

The extension will automatically apply styles to matching domains.
